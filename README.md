# Week2Day5
Getwash 🚗
​Getwash — это современная экосистема для автоматизации процессов автомойки, онлайн-записи клиентов и управления бизнес-процессами. Проект решает проблему очередей для автовладельцев и предоставляет удобную CRM-систему для владельцев бизнеса.
​✨ Основной функционал
​📱 Для клиентов (Веб / Mobile)
​Интерактивная карта и поиск: Поиск ближайших доступных автомоек.
​Умный конструктор записи: Выбор даты, времени, свободного бокса и типа автомобиля (седан, кроссовер, SUV).
​Калькулятор услуг: Выбор категорий (экспресс, комплекс, химчистка, полировка) с автоматическим расчетом финальной стоимости и времени работы.
​Личный кабинет: История заказов, управление своим автопарком, привязка карт и бонусный баланс.
​Уведомления: PUSH, SMS или Telegram-уведомления о статусе заказа и напоминания о записи.
